# Minecraft 服务器服主教程

欢迎来到 **MHST**（Minecraft Herra Server Tutorial）——一个为Minecraft服务器运维者打造的完整教程平台。

## 🎮 教程概述

本教程致力于帮助服主掌握现代Minecraft服务器架构，内容涵盖：

- **单端服务器** — 独立服务器的搭建与优化
- **群组架构** — 多服务器集群管理方案
- **Velocity代理** — 高性能服务器代理工具
- **Minecraft 1.21** — 最新版本特性与配置

无论你是初入门的新手服主，还是想升级现有架构的资深运维，这里都能找到实用的教程和最佳实践。
